Privacy Policy for Trend YJKA

Trend YJKA is a personal analytics and productivity tool used solely by the owner of this TikTok developer account.

Data Access

Trend YJKA uses the official TikTok API to access basic profile information and performance analytics such as video views, likes, audience behavior, and engagement statistics.

Data Usage

All data collected is used exclusively to analyze and improve content performance for the owner's personal TikTok account. No information is shared, sold, or distributed to third parties.

Data Storage

Any data retrieved through TikTok's API is stored privately and securely. Only the account owner has access to this data.

Access Tokens

TikTok access tokens are securely stored and used only for authentication and official API communication. The owner can revoke API access from TikTok at any time.

Contact

If you have questions about this policy, please contact the owner of this GitHub account.
